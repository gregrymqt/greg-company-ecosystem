# 📁 styles/

**Objetivo:** Gerenciar a aparência global e estrutural do painel Admin.

**O que colocar aqui:**
- `index.css`: Estilos fundamentais ou carregamento da biblioteca de classes utilitárias (ex: Tailwind).
- Variáveis de cores voltadas para dashboards e gráficos.

**Regras:**
- Componentes isolados devem conter o próprio estilo (CSS Modules) em suas respectivas pastas. Não jogue todas as regras CSS da aplicação aqui.
