# 📁 assets/

**Objetivo:** Armazenar recursos estáticos e arquivos de mídia específicos do Painel Administrativo.

**Exemplos do que colocar aqui:**
- Logotipos do backoffice
- Ícones customizados de menus (dashboards, relatórios)
- Imagens genéricas para placeholders

**Regras:**
- Arquivos muito pesados devem ser otimizados, pois embora seja um admin, a performance de carregamento ainda importa.
- Não coloque código React ou CSS aqui.
