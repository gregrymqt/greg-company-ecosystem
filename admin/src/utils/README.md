# 📁 utils/

**Objetivo:** Agrupar lógicas puras e ajudantes focados em conversões e cálculos.

**Exemplos (No contexto do Admin):**
- Funções numéricas complexas para cálculos de BI e comissões.
- Formatadores de moeda pesados.
- Scripts manipuladores de data (ex: `moment` ou `date-fns` wrappers) para calcular intervalos em relatórios.

**Regras:**
- Devem ser funções puras. Sem efeitos colaterais visuais ou de rede.
