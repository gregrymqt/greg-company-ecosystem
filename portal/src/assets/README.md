# 📁 assets/

**Objetivo:** Armazenar recursos estáticos e arquivos de mídia.

Este diretório contém todos os arquivos que não são código-fonte e que são importados diretamente pelos componentes da aplicação.

**Exemplos do que colocar aqui:**
- Imagens (`.png`, `.jpg`, `.svg`, `.webp`)
- Ícones personalizados
- Fontes tipográficas locais
- Arquivos de mídia (áudios curtos, animações Lottie)

**Regras:**
- Não coloque código React, TypeScript ou CSS aqui.
- Otimize as imagens antes de adicioná-las para não pesar o bundle da aplicação.
